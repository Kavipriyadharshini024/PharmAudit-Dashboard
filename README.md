# PharmaAudit Dashboard

PharmaAudit is a **Pharmacy Billing and Security Audit Dashboard** developed to analyze pharmacy transaction data and detect irregularities using data analytics techniques. This project demonstrates how data-driven auditing can help improve transparency, accuracy, and security in pharmacy billing systems.

## Project Overview

Pharmacy management systems handle a large number of daily transactions involving medicines, patient information, and billing records. Errors or irregularities in these systems can lead to financial loss or security issues.  

The **PharmaAudit Dashboard** is designed to analyze pharmacy billing datasets and identify anomalies such as incorrect billing calculations, unusual discounts, and unauthorized system access.

The dashboard provides a structured way to perform **billing audits, security audits, statistical analysis, and generate audit reports**.

## Features

- Upload pharmacy billing datasets (CSV or Excel)
- Perform automated billing audit checks
- Detect abnormal discounts and calculation errors
- Security audit for user access and prescription compliance
- Visual statistical analysis using charts
- Generate audit reports summarizing findings

## Dashboard Modules

### 1. Data Input
Allows users to upload pharmacy transaction datasets or manually enter billing records.

### 2. Billing Audit
Analyzes financial transactions to identify billing inconsistencies such as:
- Incorrect total calculations
- Duplicate bill IDs
- High discount percentages
- Overbilling or abnormal charges

### 3. Security Audit
Checks system usage and access control issues such as:
- Unauthorized user roles
- Medicines issued without prescription
- Missing batch verification
- Suspicious access patterns

### 4. Statistical Analysis
Provides graphical insights into transaction data including:
- Revenue patterns
- Medicine distribution
- Discount trends
- Transaction activity

### 5. Audit Report
Generates a summary of audit findings including detected anomalies and an overall audit score.
